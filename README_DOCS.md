## 🆘 Troubleshooting Quick Tips

| Issue | Solution |
|-------|----------|

| Dark mode doesn't work | Check toggle in sidebar |
| Table looks wrong | Try responsive design (zoom out) |
| Data disappears | Refresh page (wait for load) |
